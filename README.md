# Proyecto - Data Science II (Coderhouse)

## Análisis de Ventas en Ferreterías

---

## Contexto

Este proyecto se desarrolla en el marco del curso Data Science II de Coderhouse.

Se trabaja con un dataset de ventas de una red de ferreterías, con el objetivo de comenzar a aplicar herramientas de análisis de datos sobre un caso de negocio.

---

## Objetivo

Realizar un análisis exploratorio inicial del dataset para comprender el comportamiento de las ventas y detectar patrones relevantes que puedan ser profundizados en futuras etapas.

---

## Descripción del Dataset

El dataset contiene registros de transacciones de ventas. Cada fila representa una operación e incluye información como:

* Producto
* Categoría
* Medio de pago
* Sucursal
* Fecha de la venta
* Monto

---

## Preguntas / Hipótesis

A partir del análisis se busca responder las siguientes preguntas:

* ¿Qué categorías de productos concentran la mayor cantidad de ventas?
* ¿El medio de pago influye en el monto de la compra?
* ¿Existen diferencias entre sucursales en volumen de ventas o ticket promedio?

---

## Análisis realizado

* Limpieza básica de datos (duplicados, formatos)
* Exploración de variables principales
* Visualización de distribuciones y comparaciones

---

## Insights preliminares

* Las categorías de herramientas muestran una mayor participación en las ventas
* Se observan diferencias en el ticket promedio según el medio de pago
* Algunas sucursales concentran mayor volumen de operaciones que otras
